# 🔧 Digital Logic Gates — Verilog + EDA Playground

A clean collection of fundamental digital logic gates implemented in Verilog and verified using EDA Playground.

---

## 📌 Overview

This repository showcases core digital building blocks with simple, readable Verilog code and matching testbenches. Each design is simulated with waveform output for clarity and validation.

---

## 🧪 Simulation Details

* Platform: **EDA Playground**
* Simulator: *Icarus Verilog (recommended)*
* Waveforms: `.vcd` dump enabled for all designs

---

## 📁 Project Structure

```
├── *.v                     // Gate implementations
├── *tesbench_xxyyzz.v     // Testbenches
└── dump.vcd               // Waveform output
```

## 🚀 Running the Designs

1. Open any simulation link above
2. Review or modify the code
3. Run the simulation
4. Inspect waveforms for output behavior
